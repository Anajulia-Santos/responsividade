Responsividade





O que é?


Site Responsivo

DEFINAÇÃO: É uma técnica de estruturação HTML e CSS que resulta na adaptação do site ao browser do usuário.
- Dessa forma, o layout fica mais fluido e agradável. Isso impacta diretamente na usabilidade da aplicação, aumentando as visualizações e o tempo de permanência do usuário.
- O design responsivo é um método que permite ajustar o conteúdo e o design geral do seu site dependendo do dispositivo que o usuário usa para visualizá-lo. Isso significa que é importante que seu site seja responsivo para que fique bem em qualquer dispositivo.

- Responsive Design foi um termo criado por Ethan Marcotte para se referir à prática de ajustar automaticamente o conteúdo de um site à tela em que ele será exibido. Afinal, quando o desktop deixou de ser a fonte principal de conexão à internet, a nova concepção foi essencial.

Origem

"Em vez de adaptar designs desconectados a cada um de um número cada vez maior de dispositivos da web, podemos tratá-los como facetas da mesma experiência. Podemos projetar para uma experiência de visualização ideal, mas incorporar tecnologias baseadas em padrões em nossos projetos para torná-los não apenas mais flexíveis, mas mais adaptáveis ​​à mídia que os processa. Resumindo, precisamos praticar o design responsivo da web."

Ethan Marcotte


Mini Linha do Tempo


Início do Web Design

No início da web, as páginas eram projetadas apenas para monitores de computadores, com layouts fixos. Isso tornou a visualização em dispositivos menores, como celulares, desafiadora.

A Era dos Sites Móveis

Com o crescimento dos dispositivos móveis, surgiram sites móveis separados, projetados especificamente para smartphones. Embora fossem uma solução, exigiam duplicação de esforço para manter dois tipos de sites.

A Adoção Generalizada

O design responsivo logo se tornou uma prática padrão. O Bootstrap, um popular framework front-end, foi lançado em 2011, simplificando a criação de sites responsivos.

O Futuro da Responsividade

A responsividade continua a evoluir com a proliferação de dispositivos. O design adaptativo, imagens otimizadas para web e tecnologias como CSS Grid e Flexbox estão moldando o futuro do design web responsivo.
